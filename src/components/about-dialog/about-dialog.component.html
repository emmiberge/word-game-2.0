How to play

Find groups of four items that share something in common.

  • When you have chosen four words, select 'Submit' to make a guess.
  • If you make four mistakes, you lose!

Examples of categories:

  • FURNITURE: Chair, Table, Lamp, Sofa
  • SUPER ___ : Nova, Hero, Market, Star

Each puzzle has exactly one solution. Some words might seem to belong to multiple categories, you might need to think about them for a while!

Each group has its own color, which you can see once you find a particular group.
